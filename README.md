# CPU-Scheduler
CPU Scheduling Algorithms Visualizer
An interactive web application that simulates and visualizes various CPU scheduling algorithms used in operating systems. This tool helps students understand scheduling behavior through dynamic Gantt charts, process tables, and performance metric analysis.

🚀 Features

✅ Multiple Scheduling Algorithms:
FCFS (First Come First Serve)
SJF (Shortest Job First)
SRTF (Shortest Remaining Time First)
Priority Scheduling (Preemptive & Non-Preemptive)
Round Robin
Round Robin with Priority

📊 Dynamic Gantt Chart Visualization

📋 Automatic Process Result Table

📈 Performance Metrics Calculation:
Average Waiting Time
Average Turnaround Time
Average Response Time
CPU Utilization
Throughput

📘 Step-by-step Metric Formula Guide

🎨 Clean and Responsive UI

⚡ Fully client-side (HTML, CSS, JavaScript)

🛠 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Canvas API (for Gantt Chart)
